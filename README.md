# stockskill

This repository publishes `zhaogeskill`, a ZhaoGe-style trading-learning skill for stock, ETF, leveraged ETF, crypto-related equity, and options analysis.

## Layout

- `SKILL.md`: high-level trigger conditions, safety rules, and answer formats
- `references/framework.md`: full methodology and playbook
- `agents/openai.yaml`: agent metadata and default prompt

## Notes

- This is a learning and review framework, not a signal service.
- Current prices, news, earnings, macro events, and options data should always be verified before use.
